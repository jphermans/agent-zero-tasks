# Agent Zero Tasks

Deze repository bevat geautomatiseerde taken die door Agent Zero worden uitgevoerd.

## 📋 Overzicht van Taken

### 1. 🧩 Dagelijkse Sudoku
**Script:** `sudoku/daily_sudoku.py`
**Schedule:** Elke dag om 08:00 (Europe/Brussels)

**Beschrijving:**
- Genereert een nieuwe sudoku puzzle elke ochtend
- Stuurt de oplossing van de vorige dag via Telegram
- Start een Cloudflare tunnel voor webtoegang
- Stuurt een interactieve link naar de sudoku

**Kenmerken:**
- Interactieve HTML sudoku voor iPhone
- Automatische voortgang opslag (localStorage)
- Controleer en Reset knoppen
- Geen oplossing-knop (oplossing komt de volgende dag)

### 2. 🔍 Dagelijkse Woordzoeker
**Script:** `woordzoeker/generate_puzzle.py`
**Schedule:** Elke dag om 12:00 (Europe/Brussels)
**GitHub Pages:** https://jphermans.github.io/woordzoeker/

**Beschrijving:**
- Genereert een nieuwe Nederlandstalige woordzoeker elke dag
- 10 Nederlandse woorden per puzzle
- Woorden uit verschillende thema's (dieren, eten, strand, etc.)
- Stuurt Telegram notificatie met link

**Kenmerken:**
- Interactieve HTML woordzoeker
- Gevonden woorden blijven gekleurd (groen) op het bord
- Woorden worden doorstreept in de lijst
- Automatische voortgang opslag (localStorage per dag)
- Hint functie beschikbaar
- Reset voortgang knop
- Woorden kunnen horizontaal, verticaal en diagonaal verborgen zijn

### 3. 🏆 Daily OpenRouter Cheapest Coding Models Reporter

### 4. 📊 Invoice Checker Daily
**Script:** `invoice/invoice_checker.py`
**Schedule:** Driemaal daags om 08:00, 14:00, 20:00 (Europe/Brussels)

**Beschrijving:**
- Verbindt met Fastmail IMAP
- Zoekt naar emails met factuur-keywords (invoice, faktuur, rechnung, etc.)
- Extraheert PDF bijlagen en analyseert deze
- Stuurt Telegram notificatie met samenvatting

**Kenmerken:**
- Herkent bedragen, verzenders en vervaldatums
- Groepeert facturen per afzender
- Slaat verwerkte emails op om duplicaten te voorkomen

**Configuratie:**
- Secrets: FASTMAIL_USER, FASTMAIL_APP_PASSWORD, TELEGRAM_BOT_TOKEN, TELEGRAM_USERID

## 🔧 Technische Details

### Cloudflare Tunnel
De sudoku taak gebruikt een Cloudflare tunnel:

```
┌─────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   iPhone    │ ──▶ │ Cloudflare      │ ──▶ │ Lokale Server   │
│   (Safari)  │     │ Tunnel          │     │ (Docker:8765)   │
└─────────────┘     └─────────────────┘     └─────────────────┘
```

### GitHub Pages
De woordzoeker wordt gehost op GitHub Pages:
- URL: https://jphermans.github.io/woordzoeker/
- Automatische updates via dagelijkse push

- TELEGRAM_USERID
- FASTMAIL_USER
- FASTMAIL_APP_PASSWORD
- GITHUB_TOKEN
- OpenRouter API key (extern bestand)
