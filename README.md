# Agent Zero Tasks

Deze repository bevat geautomatiseerde taken die door Agent Zero worden uitgevoerd.

## 📋 Overzicht van Taken

### 1. 🧩 Dagelijkse Sudoku
**Script:** `sudoku/daily_sudoku.py`
**Schedule:** Elke dag om 08:00 (Europe/Brussels)

**Beschrijving:**
- Genereert een nieuwe sudoku puzzle elke ochtend
- Stuurt de oplossing van de vorige dag via Telegram
- Start een Cloudflare tunnel voor webtoegang
- Stuurt een interactieve link naar de sudoku

**Kenmerken:**
- Interactieve HTML sudoku voor iPhone
- Automatische voortgang opslag (localStorage)
- Controleer en Reset knoppen
- Geen oplossing-knop (oplossing komt de volgende dag)

---

### 2. 🏆 Daily OpenRouter Cheapest Coding Models Reporter
**Script:** `openrouter/openrouter_reporter.py`
**Schedule:** Elke dag om 09:00 (Europe/Brussels)

**Beschrijving:**
- Haalt de goedkoopste coding modellen op van OpenRouter
- Genereert een mooi HTML rapport
- Stuurt het rapport via email
- Stuurt een samenvatting via Telegram

**Kenmerken:**
- Top 5 goedkoopste coding modellen
- Account credits tracking
- Professionele HTML email met styling

**Configuratie:**
- API key moet in `/a0/usr/workdir/openrouter_api_key.txt` staan

### 3. 🧹 Daily Task Cleanup
**Schedule:** Elke dag om 17:00 (Europe/Brussels)

**Beschrijving:**
